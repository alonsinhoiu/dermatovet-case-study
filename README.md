# DermatoVet

## Veterinary Dermatology Website and Appointment Automation

DermatoVet is a professional website and automated appointment-request workflow developed for a veterinary dermatology service. It gives clients a clear way to request appointments while helping the veterinary professional organize and confirm them with less manual administrative work.

🌐 [Visit DermatoVet](https://dermatovet.org)

## The problem

Managing appointment requests manually can require repeated messages, copying information between tools, checking availability, and creating calendar events individually. This takes time and increases the possibility of missed or disorganized requests.

## The solution

I developed the public website and connected its appointment form to an automated workflow. Requests are recorded and communicated to the veterinary professional, who can review availability before confirming an appointment. Once confirmed, the corresponding event is created in Google Calendar.

## Workflow

1. A client submits an appointment request through the website.
2. The system sends an automatic notification.
3. The request is recorded in a spreadsheet for review.
4. The veterinary professional reviews and confirms availability.
5. A confirmed appointment creates an event in Google Calendar.
